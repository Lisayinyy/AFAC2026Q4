# 官方资料清单（实测解读）

大赛提供的原始资料及其性质（放置于 `~/Desktop/AFAC2026/`，不入库）。

## 1. `股票样本.xlsx` — 预测目标池
- 100 只**沪市**股票：`股票代码`(如 `603316.SH`) + `股票简称`(诚邦股份)。
- 全部 `.SH` 后缀。这是 A/B 榜要出结果的标的池。

## 2. `AFAC2026赛题一训练数据.xlsx` — 唯一真实数据（样例集）
- 单只股票 `603997.SH`、单日 `20260507`，4937 个十档快照 tick，65 列。
- 对应官方"样例集：股票数量 1、时间段 2026/05/07"。
- 每行一个快照，含 price/volume/amount/transactions（累计）、totalbid/askvolume、bigordervolume，及 `bids`/`asks`（10 档 JSON，每档 `{price, volume, order:[{volume}...], bigOrderPercent}`）。
- **实测**：`order` 拆单数组**仅一档(best bid/ask)提供**，2-10 档只有 price/volume。
- 用途：开发特征工程的起步参考。**不含** 100 只目标股票的数据。

## 3. `赛题一A榜提交样例/` — 提交格式样例
- `pattern_reco.csv`(20 行) + `predict_result.csv`(1098 行)，均**每 (stock_code, transaction_date) 一行**。
- ⚠ 是**格式演示数据**：用的是深市等其它股票、多个日期(0421–0521)，且 capital_type↔capital_intention 呈 1:1 假映射（散户↔T0/游资↔买入/量化↔卖出），**非真实标签**。
- 价值：确定确切列名、代码格式(.SH/.SZ)、日期(YYYYMMDD)、以及官方词表。

## 4. `submit/` — 提交骨架模板
- `pattern_reco.csv` + `predict_result.csv`，各 20 行，6XXXXX 占位代码，日期 20260608(A榜首日)。
- 填好后打包 `submit.zip`。

## 官方词表（本仓库已严格对齐）

**pattern_type（12 类）** + 标准说明：

| pattern_type | pattern_explanation |
|---|---|
| 大单吸筹 | 资金大笔挂单买入，短时间内集中扫货 |
| 对倒拉升 | 通过自买自卖制造成交量放大假象，吸引跟风盘 |
| 压单吸货 | 在卖盘挂大单压制股价，同时在低位悄悄吸纳筹码 |
| 尾盘突袭 | 在下午2点半之后集中拉升，制造假象和典型技术形态 |
| 集合竞价异动 | 在集合竞价阶段大幅拉高或压低，影响开盘价 |
| 分时脉冲 | 短时间内快速拉升后迅速回落，试探上方抛压 |
| 量化T0 | 利用Level2行情数据高频捕捉盘口价差，自动完成T0回转 |
| 日内套利 | 资金在一定价格区间来回高抛低吸 |
| 连续小单推升 | 用大量小单持续买入缓慢推高股价，避免引起注意 |
| 盘中诱多 | 盘中快速拉升吸引跟风后迅速砸盘出货 |
| 涨停板打开 | 封涨停后反复打开，制造换手假象引诱散户接盘 |
| 散户博弈 | 全天成交量低迷，缺乏主力引导，散户之间互相换手 |

**capital_type（3 类）**：散户 / 游资 / 量化
**capital_intention**：买入 / 卖出 / T0交易（官方样例仅见此三类；赛题目标页另提"中性"，本仓库保留为合法值）

## 结论 / 数据瓶颈
- 提交粒度 = 每 (股票, 交易日) 一行，管线已符合。
- **最大瓶颈**：官方仅给 1 只样例股票的快照。要真正参赛，需自行获取 100 只目标股票每个交易日的 Level-2 快照（规则允许淘宝/闲鱼/百度网盘等渠道）。特征工程与模型框架已就绪，数据到位即可批量出结果。
